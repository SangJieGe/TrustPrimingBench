# TrustPrimingBench

TrustPrimingBench is an open research project focusing on prompt injection security for LLM agents.

Current research topics include:

- Trust Priming Attack
- Future Trust Manipulation
- Prompt Injection
- Indirect Prompt Injection
- Agent Security
- Trust Source Modeling
- Authority Binding
- Benchmark Design

This repository is under active development.

## Roadmap

- Build attack taxonomy
- Design benchmark datasets
- Evaluate major LLMs
- Publish reproducible experiments
- Release research papers
- Open community contributions
